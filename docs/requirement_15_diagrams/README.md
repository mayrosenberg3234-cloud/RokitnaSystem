# תרשימים מעודכנים — דרישה 15

בתיקייה זו נמצאים התרשימים המעודכנים של חלק ג', המבוססים בדיוק על הקוד שבמערכת.

## קבצים להגשה

* `Requirement15_UpdatedDiagrams_All.pdf` — שלושת התרשימים בקובץ PDF אחד.
* `Requirement15_ClassDiagram_Updated.pdf` — Class Diagram מעודכן.
* `Requirement15_SequenceDiagram_Updated.pdf` — Sequence Diagram מעודכן.
* `Requirement15_StateDiagram_Updated.pdf` — State Diagram מעודכן.

לנוחות, כל תרשים קיים גם בפורמט PNG ו-SVG. קבצי `.dot` הם קובצי מקור טקסטואליים של התרשימים.

## סימון חלק ג'

הירוק בתרשימים מסמן תוספות/עדכונים שנעשו בחלק ג' כדי לשמור על התאמה מלאה בין הקוד, ה-Sequence וה-State Diagram.

## הערה להגשה

התרשימים הללו מחליפים את גרסת חלק ב' עבור דרישה 15. הם מציגים רק את המחלקות הנדרשות לתהליך הממומש, כדי לשמור על קריאות ולהימנע ממחלקות שאינן קיימות במימוש של דרישה זו.
